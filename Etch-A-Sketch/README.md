# Etch-A-Sketch
A simple Etch-A-Sketch drawing app built with Pythonâ€™s `turtle` graphics module. Use your keyboard to move the turtle and draw on the screen!

## How to Use

- **W**: Move forward
- **S**: Move backward
- **A**: Rotate left
- **D**: Rotate right
- **C**: Clear the sketch

Click anywhere on the window to exit.
